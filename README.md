## 💡 How It Works 🧠

The application utilizes advanced image processing to identify axes and data points. It traces lines and labels to determine coordinate values. It relies on Tesseract to read text labels from the graph axes. Subpixel detection ensures the accuracy of point locations within the chart.

## 🆘 Troubleshooting Common Issues 🛠️

*   Application does not open: Ensure your antivirus software does not block the file. Sometimes strict security settings flag new applications.
*   Formatting issues: Make sure your input image is clear and high-resolution. Blurry images reduce the accuracy of the extraction process.
*   Chart not detected: Verify the image shows a clear coordinate system. The software requires distinct X and Y axes to perform calibration.
*   Browser window remains blank: Wait a few seconds for the local server to start. If the page does not load after thirty seconds, close the application and try to launch it again.

## 📈 Tips for Best Results 🌟

1. Use images with white backgrounds to improve contrast.
2. Ensure the axes labels are horizontal or vertical. Rotated text causes issues for the recognition engine.
3. Crop the image to focus on the graph area. Extra white space around the edges can distract the software.
4. Use standard image file formats like JPEG, PNG, or TIFF for consistent results.
5. Keep your image orientation upright. Do not rotate images before uploading.
